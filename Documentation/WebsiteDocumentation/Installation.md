## Installation

You may install this as part of [OceanKit](https://github.com/JeffreyEarly/OceanKit), or directly from here using the command-line,
```
https://github.com/satmapkit/AlongTrackSimulator
```
to clone the repo, and then installing package from within MATLAB
```matlab
mpminstall("local/path/to/AlongTrackSimulator", Authoring=true); 
```
with authoring enabled.
