#  WVAlongTrackObservingSystem

create a new along track observing system


---

## Declaration
```matlab
 self = WVAlongTrackObservingSystem(model,alongTrackGroup)
```
## Parameters
+ `model`  the WVModel instance
+ `alongTrackGroup`  name of the observing system

## Returns
+ `self`  a new instance of WVAlongTrackObservingSystem

## Discussion

  This class is intended to be subclassed, so it generally
  assumed that this initialization will not be called directly.
