#  computeNodalPeriod

Computes the nodal precession period from orbital parameters


---

## Declaration
```matlab
 T_nodal = computeNodalPeriod(a, e, i)
```
## Parameters
+ `a`  semi-major axis [km]
+ `e`  eccentricity (0 for circular, >0 for elliptical)
+ `i`  inclination [degrees]

## Returns
+ `T_nodal`  nodal period [s]
