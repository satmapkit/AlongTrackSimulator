#  computeNodalPeriod

COMPUTE_NODAL_PERIOD Computes the nodal precession period


---

## Discussion

  Inputs:
    a  - Semi-major axis (km)
    e  - Eccentricity
    i  - Inclination (degrees)
    mu - Gravitational parameter (default: Earth's, 398600.4418 km^3/s^2)
 
  Output:
    T_nodal - Nodal period (seconds)
